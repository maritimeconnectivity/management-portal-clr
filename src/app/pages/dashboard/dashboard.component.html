MCP components

Identity Registry
Features
  • Identity Management: Provides each maritime entity a unique Maritime Resource Name (MRN).
  • Public Key Infrastructure (PKI): Ensures entities have a secure cryptographic identity, including key pairs and certificates.
  • Authentication: Supports authorization for secure integration, using standards like OAuth 2.0/OpenID Connect.

Service Registry
Features
  • Service Discovery: Enable end-users to discover these services. Users can search for services based on keywords, organizations, locations, or a combination of criteria.
  • Service Management: Supports the management of service information for service providers.
Documentation Dev Docs Repo
Maritime Messaging Service
Features
  • MRN addressed messaging: Uses the Maritime Resource Name (MRN) as a standardized endpoint address, enabling consistent and precise delivery of messages.
  • Broadcast (subject-cast) messaging: Enable broadcast messages to users who have subscribed to a specific subject tag.
  • Seamless Information Transfer: Enables transparent communication across various connection types.
