How to search service?

Service Registry provides a service search with parameters defined in IEC 63173-2 SECOM standard. Please choose parameter from Add query elements and enter value you want to search and press Search button.

Geographical search: You can also do a geometry search by clicking icons on the left side.
So you can define an area of interest together with the search parameters.
